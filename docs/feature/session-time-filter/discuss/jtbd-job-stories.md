# Job Stories: Session Time Filter

## Job 1 — Focus on what's happening now

**When** I open the Sessions view and see a long list of sessions accumulated over days,
**I want to** filter down to only the sessions that are active right now,
**so I can** immediately understand my current Claude Code workload without scanning past noise.

**Dimensions:**
- **Functional**: Reduce the session list to currently active sessions
- **Emotional**: Feel in control, not overwhelmed by historical clutter
- **Social**: N/A (single-user desktop app)

## Job 2 — Review recent activity

**When** I return to Norbert after a break (lunch, meeting, overnight),
**I want to** see sessions from a specific recent time window (last 15 min, last hour, last 24 hours),
**so I can** quickly catch up on what happened while I was away without digging through the full history.

**Dimensions:**
- **Functional**: Scope the session list to a recency window
- **Emotional**: Feel confident I haven't missed anything important
- **Social**: N/A

## Job 3 — Understand session volume over time

**When** I'm trying to understand my usage patterns,
**I want to** toggle between time windows and see how many sessions fall into each,
**so I can** get a sense of my Claude Code activity density without leaving the sessions view.

**Dimensions:**
- **Functional**: See session count per time window as feedback
- **Emotional**: Feel informed about usage patterns
- **Social**: N/A
